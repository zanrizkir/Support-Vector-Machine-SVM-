# Support Vector Machine (SVM) with Python

Program ini merupakan implementasi algoritma **Support Vector Machine (SVM)** menggunakan Python dan library **Scikit-Learn** untuk melakukan klasifikasi pada dataset **Breast Cancer Wisconsin**.

## Fitur
- Import dan eksplorasi dataset
- Pembagian data training dan testing
- Pelatihan model SVM
- Prediksi data testing
- Evaluasi model menggunakan:
  - Confusion Matrix
  - Classification Report

## Library yang Digunakan
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-Learn

## Cara Menjalankan
1. Install library yang diperlukan.
2. Jalankan file notebook (`.ipynb`) menggunakan Jupyter Notebook atau Visual Studio Code.
3. Ikuti setiap sel kode secara berurutan untuk melihat proses pelatihan dan evaluasi model.
